BTracT Dashboard



Graphs

The graphs in this dashboard will provide more information when you mouse-over elements of the graph. Most of the graphs are interactive. By using global filters or clicking bars of these graphs expands to specific details.

The image of the graph may be downloaded in PNG or other formats as provided in the graph menu. To save the plot, right click on it and then select save picture as . In the resulting dialog box, select the location to save the image and type in the image name in the File name box.

Data Tables

Data tables has many useful options such as search and filters (just below the table headers) as describe below;

  • Search Box: Located at top-right corner of data table. Use to search values within the table
  • Table Filters: Filter within column. Multiple filters are posible for a fine view
  • Page Length: Shows number of entries to display in the table
  • Use the Download button at top-right corner of table to download data. Selecting specific rows, subsets the data downloaded.
