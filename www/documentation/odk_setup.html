What you need

  • Android mobile phone or tablet

  • Zebra ZQ520 mobile printer

Install apps

On your mobile phone, go to play store and download

  • ODK collect
  • ODK Sensors Framework
  • ODK printer driver (contact admin)

Configuration

Open ODK collect


Go to the Menu (top right corner)

a. Login details

Go to General settings as indicated in the fig


Select Server settings

Replace the default information with your login info


URL: https://ona.io

Username: your username

Password: your password

b. Form management


Go to the MENU


Select auto send - set to wifi or data

End of Configuration
